# Табель учёта рабочего времени — WordPress Plugin

## Установка

1. Загрузите `tabel-ucheta.zip` в WordPress:
   - **Плагины → Добавить новый → Загрузить плагин**
   - Выберите `tabel-ucheta.zip` и нажмите "Установить"
2. Активируйте плагин
3. Создайте страницу и добавьте шорткод: `[tabel_ucheta]`
4. Откройте эту страницу — появится форма входа

## Вход по умолчанию

- **Логин:** `Frazy`
- **Пароль:** `FrazyAmp`
- Это суперадминистратор с полными правами

## Настройка Telegram-уведомлений

В `wp-admin` добавьте опции:
```php
update_option('tabel_telegram_token', 'ВАШ_BOT_TOKEN');
update_option('tabel_telegram_chat_id', 'ВАШ_CHAT_ID');
```

## Структура файлов

```
tabel-ucheta/
├── tabel-ucheta.php          # Главный файл плагина
└── includes/
    ├── database.php           # Таблицы БД, хелперы расчётов
    ├── auth.php               # Авторизация (сессии PHP)
    ├── api.php                # Все API-маршруты (AJAX)
    ├── export.php             # Экспорт Excel (HTML-таблицы)
    ├── translations.php       # Переводы (RU/KG/EN)
    └── templates/
        ├── login.php          # Форма входа
        └── app.php            # Основное SPA-приложение
```

## Особенности

- Авторизация через PHP-сессии (не WordPress-пользователи)
- Мульти-база данных с правами доступа
- Два типа расчётов: ПКС (педагоги) и Сотрудники
- Экспорт в Excel без внешних PHP-библиотек
- 3 языка: Русский, Кыргызский, English
